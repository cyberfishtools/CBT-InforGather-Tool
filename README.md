 ## How to install: 
 1. Download
 - clone through termial :
  ``` 
  git clone https://github.com/cyberfishtools/CBT-InforGather-Tool.git
  ```
  2. Run
  -in terminal
  ```
  chmod +x CBT-InforGather-Tool.sh
  ./CBT-InforGather-Tool.sh
  ```
